Программа для замены чисел массива больших n на среднее арифметическое

Как работает программа:

1. При нажатии кнопки "Сгенерировать массив" вызывается метод GenerateButton_Click

2. Создаётся объект Random для генерации случайных чисел

3. Создаётся массив array размером 10 элементов

4. В цикле каждый элемент массива заполняется случайным числом от 1 до 99

5. Массив преобразуется в строку через запятую и выводится в ArrayTextBox

6. Пользователь вводит число n в поле NTextBox

7. При нажатии кнопки "Заменить числа > n" вызывается метод ReplaceButton_Click

8. Значение n считывается из текстового поля

9. Вычисляется сумма всех элементов массива в переменной sum

10. Среднее арифметическое вычисляется делением sum на длину массива

11. Результат деления преобразуется в тип double для точности

12. В цикле проверяется каждый элемент массива

13. Если элемент больше n, он заменяется на среднее арифметическое (преобразованное в int)

14. Изменённый массив выводится в поле ResultTextBox

15. При ошибке ввода выводится сообщение "Ошибка ввода"
