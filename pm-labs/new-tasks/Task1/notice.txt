Программа для вычисления суммы нечётных чисел из массива

Как работает программа:

1. Пользователь вводит числа через запятую в текстовое поле InputTextBox

2. При нажатии кнопки "Вычислить сумму нечётных" срабатывает метод CalculateButton_Click

3. Входная строка разбивается на части по запятой с помощью метода Split

4. Создаётся массив numbers и каждая часть строки преобразуется в число через int.Parse

5. Формируется строка arrayText для отображения массива чисел через запятую

6. Массив выводится в текстовое поле ArrayTextBox

7. Проходим по массиву и проверяем каждое число на нечётность через остаток от деления на 2

8. Если число нечётное (остаток не равен 0), добавляем его к переменной sum

9. Итоговая сумма выводится в текстовое поле ResultTextBox

10. Если возникает ошибка при вводе, выводится сообщение "Ошибка ввода"
