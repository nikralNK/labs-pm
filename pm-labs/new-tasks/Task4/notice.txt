Программа для объединения двух массивов

Как работает программа:

1. Пользователь вводит первый массив чисел через запятую в поле Array1TextBox

2. Пользователь вводит второй массив чисел через запятую в поле Array2TextBox

3. При нажатии кнопки "Объединить массивы" вызывается метод MergeButton_Click

4. Каждая входная строка разбивается на части методом Split(',')

5. Создаются два массива array1 и array2 нужного размера

6. Значения из строковых массивов преобразуются в числа и записываются в array1 и array2

7. Создаётся новый массив mergedArray размером равным сумме длин array1 и array2

8. В первый цикл копируются все элементы из array1 в начало mergedArray

9. Во втором цикле копируются все элементы из array2 в конец mergedArray

10. Индекс для второго массива вычисляется как array1.Length + i

11. Объединённый массив преобразуется в строку с разделителем ", "

12. Результат выводится в поле ResultTextBox

13. При ошибке ввода выводится сообщение "Ошибка ввода"
