Программа для динамического создания элементов интерфейса

Как работает программа:

1. В интерфейсе размещены три радиокнопки для выбора типа элемента: TextBox, Button и Label

2. По умолчанию выбрана радиокнопка TextBox (свойство IsChecked="True")

3. При нажатии на кнопку "Создать элемент" вызывается метод CreateElement_Click

4. Метод проверяет какая радиокнопка выбрана с помощью условия if-else if

5. Если выбран TextBoxRadio, создаётся новый объект TextBox с шириной 100, высотой 25 и отступом 5

6. Если выбран ButtonRadio, создаётся новый объект Button с текстом "Кнопка"

7. Если выбран LabelRadio, создаётся новый объект Label с текстом "Текст"

8. Созданный элемент добавляется в ElementsPanel через метод Children.Add()

9. ElementsPanel имеет Orientation="Horizontal", поэтому каждый новый элемент размещается правее предыдущего

10. Свойство Margin = new Thickness(5) создаёт отступ в 5 пикселей вокруг каждого элемента
