Программа для поиска максимального количества подряд идущих пробелов

Как работает программа:

1. Пользователь вводит строку в текстовое поле InputTextBox

2. При нажатии кнопки "Найти макс. пробелы" вызывается метод FindButton_Click

3. Входная строка сохраняется в переменную input

4. Создаются две переменные: maxSpaces для хранения максимального количества и currentSpaces для текущего счётчика

5. Запускается цикл, проходящий по каждому символу строки

6. Если текущий символ является пробелом, увеличиваем currentSpaces на 1

7. После увеличения проверяем, больше ли currentSpaces чем maxSpaces

8. Если больше, обновляем значение maxSpaces

9. Если символ не пробел, сбрасываем currentSpaces в 0

10. Цикл продолжается до конца строки

11. В результате в maxSpaces сохраняется наибольшее количество подряд идущих пробелов

12. Это число выводится в поле ResultTextBox

13. Например, для строки "a   b  c" результат будет 3
