Программа для подсчёта количества слов в строке

Как работает программа:

1. Пользователь вводит строку со словами в текстовое поле InputTextBox

2. Слова в строке должны быть разделены пробелами

3. При нажатии кнопки "Подсчитать слова" вызывается метод CountButton_Click

4. Входная строка сохраняется в переменную input

5. Строка разбивается на массив слов методом Split(' ')

6. В качестве разделителя используется пробел

7. Создаётся переменная count со значением 0 для подсчёта слов

8. Запускается цикл, проходящий по каждому элементу массива words

9. Для каждого элемента проверяется условие: не является ли он пустой строкой

10. Проверка words[i] != "" нужна для исключения лишних пробелов

11. Если элемент не пустой, увеличивается счётчик count

12. После завершения цикла count содержит количество непустых слов

13. Значение count преобразуется в строку и выводится в ResultTextBox

14. Например, для строки "привет мир" результат будет 2
