Программа для вставки символа 'b' после каждого символа 'a'

Как работает программа:

1. Пользователь вводит строку в текстовое поле InputTextBox

2. При нажатии кнопки "Вставить 'b' после 'a'" вызывается метод InsertButton_Click

3. Входная строка сохраняется в переменную input

4. Создаётся пустая строка result для накопления результата

5. Запускается цикл, который проходит по каждому символу входной строки

6. На каждой итерации текущий символ добавляется к строке result

7. После добавления символа проверяется условие: если символ равен 'a'

8. Если условие выполнено, к строке result добавляется символ 'b'

9. Цикл продолжается до конца строки

10. Итоговая строка с вставленными 'b' выводится в поле ResultTextBox

11. Например, строка "banana" превратится в "babnanaba"
