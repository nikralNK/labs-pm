Программа для обмена двух слов местами

Как работает программа:

1. Пользователь вводит два слова через пробел в текстовое поле InputTextBox

2. При нажатии кнопки "Поменять слова местами" вызывается метод SwapButton_Click

3. Содержимое текстового поля сохраняется в переменную text

4. Строка разбивается на массив слов с помощью метода Split(' '), где ' ' - разделитель (пробел)

5. Проверяется количество элементов в массиве words с помощью условия if (words.Length == 2)

6. Если слов ровно два, выполняется обмен значениями

7. Создаётся временная переменная temp, в которую сохраняется первое слово

8. В первый элемент массива записывается второе слово

9. Во второй элемент массива записывается значение из temp (исходное первое слово)

10. Текст в InputTextBox заменяется на новую строку, состоящую из поменянных местами слов

11. Слова объединяются через пробел с помощью оператора +
