Программа для случайного перемешивания элементов массива

Как работает программа:

1. При нажатии кнопки "Сгенерировать массив" вызывается метод GenerateButton_Click

2. Создаётся объект Random для генерации случайных чисел

3. Создаётся массив array размером 10 элементов

4. В цикле каждый элемент массива заполняется случайным числом от 1 до 99

5. Массив преобразуется в строку через запятую и выводится в ArrayTextBox

6. При нажатии кнопки "Перемешать массив" вызывается метод ShuffleButton_Click

7. Создаётся новый объект Random для перемешивания

8. Используется алгоритм перемешивания Фишера-Йетса

9. Цикл идёт с конца массива к началу (от array.Length - 1 до 1)

10. На каждой итерации выбирается случайный индекс j от 0 до i включительно

11. Элементы с индексами i и j меняются местами через временную переменную temp

12. Сначала значение array[i] сохраняется в temp

13. Затем array[i] получает значение array[j]

14. И array[j] получает значение из temp

15. Перемешанный массив преобразуется в строку и выводится в ResultTextBox
