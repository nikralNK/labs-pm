# Лабораторная работа 19 - Вычисление функции с условиями

## Описание задачи

Создано приложение для вычисления значения функции в зависимости от значения переменной x и параметра a.

## Формулы функций

### Функция 1 (при x <= 0)
```
f1(x) = x * sin(x - 1)
```

### Функция 2 (при 0 < x <= a)
```
f2(x) = (x - 1)³ + cos(x³)
```

### Функция 3 (при x > a)
```
f3(x) = √(|x|³) * sin(x³)
```

## Элементы интерфейса

**Поля ввода:**
- `txtA` - ввод значения параметра a
- `txtX` - ввод значения переменной x
- `txtY` - вывод результата (только чтение)

**Кнопки:**
- `btnCalculate` - запуск вычисления
- `btnExit` - выход из приложения

**Режим выбора функции:**
- `chkSelectFunction` - CheckBox для включения ручного выбора функции
- `grpFunctions` - GroupBox с RadioButton для выбора конкретной функции
  - `rbFunction1` - выбор первой функции
  - `rbFunction2` - выбор второй функции
  - `rbFunction3` - выбор третьей функции

## Логика работы

### Режим автоматического выбора (CheckBox не отмечен)

При нажатии кнопки "Вычислить" программа:
1. Считывает значения a и x из текстовых полей
2. Проверяет условия и выбирает функцию автоматически:
   - Если x <= 0, то используется функция 1
   - Если 0 < x <= a, то используется функция 2
   - Если x > a, то используется функция 3
3. Вычисляет результат
4. Выводит результат в поле txtY с точностью 6 знаков после запятой

### Режим ручного выбора (CheckBox отмечен)

При установке флажка "Выбрать конкретную функцию":
1. Группа с RadioButton становится активной (Enabled = true)
2. Пользователь может выбрать любую из трех функций
3. При вычислении используется выбранная функция независимо от значения x

При снятии флажка:
1. Группа с RadioButton становится неактивной (Enabled = false)
2. Возвращается автоматический режим выбора функции

## Обработчики событий

### btnCalculate_Click
Основной обработчик вычисления:
- Проверяет режим работы (автоматический или ручной выбор)
- В режиме ручного выбора проверяет, какая RadioButton выбрана
- В автоматическом режиме проверяет условия для x
- Вычисляет соответствующую функцию
- Обрабатывает ошибки ввода данных

### btnExit_Click
Закрывает приложение методом Close().

### chkSelectFunction_CheckedChanged
Управляет доступностью группы с RadioButton:
- Если CheckBox отмечен - группа активна
- Если CheckBox снят - группа неактивна

## Математические функции

Используются методы класса Math:
- `Math.Sin()` - синус
- `Math.Cos()` - косинус
- `Math.Pow(x, 3)` - возведение в степень
- `Math.Sqrt()` - квадратный корень
- `Math.Abs()` - модуль числа

## Обработка ошибок

При некорректном вводе данных (нечисловые значения) выводится сообщение "Ошибка ввода данных".
