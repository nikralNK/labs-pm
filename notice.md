# Лабораторная работа 18 - Текстовый редактор

## Описание проекта

Создан текстовый редактор с графическим интерфейсом для работы с RTF файлами.

## Структура приложения

### Основные компоненты

**RichTextBox (richTextBox1)** - основное поле для редактирования текста
- Размещен с заполнением всей формы (Dock = Fill)
- Поддерживает форматирование RTF

**MenuStrip (menuStrip1)** - главное меню приложения
- Содержит три раздела: File, Edit, Help

**ToolStrip (toolStrip1)** - панель инструментов
- Дублирует основные команды меню для быстрого доступа

**StatusStrip (statusStrip1)** - строка состояния
- Отображает имя текущего файла

**OpenFileDialog / SaveFileDialog** - диалоги открытия и сохранения файлов
- Настроены для работы с RTF файлами

## Меню File

### New (Ctrl+N)
Создает новый файл. При выполнении команды очищается текстовое поле, устанавливается имя файла по умолчанию.

### Open (Ctrl+O)
Открывает существующий файл. Показывает диалог выбора файла, после выбора загружает содержимое в RichTextBox.

### Save (Ctrl+S)
Сохраняет текущий файл. Если файл еще не сохранялся (имя = untitled.rtf), вызывается команда Save As.

### Save As
Сохраняет файл с новым именем. Показывает диалог сохранения, позволяет выбрать имя и расположение файла.

### Exit (Ctrl+Q)
Закрывает приложение.

## Меню Edit

### Cut (Ctrl+X)
Вырезает выделенный текст в буфер обмена. Использует встроенный метод RichTextBox.Cut().

### Copy (Ctrl+C)
Копирует выделенный текст в буфер обмена. Использует встроенный метод RichTextBox.Copy().

### Paste (Ctrl+V)
Вставляет текст из буфера обмена. Использует встроенный метод RichTextBox.Paste().

## Меню Help

### Contents (Ctrl+H)
Показывает справку о программе в виде MessageBox с описанием команд меню.

### About
Открывает окно "О программе" с информацией о приложении.

## Панель инструментов

Содержит кнопки для быстрого доступа к командам:
- New, Open, Save
- Cut, Copy, Paste
- Help

Все кнопки связаны с обработчиками меню, чтобы избежать дублирования кода.

## Переменные и константы

**fileName** - хранит путь к текущему файлу
**defaultFileName** - имя файла по умолчанию ("untitled.rtf")

## Работа с файлами

При загрузке приложения устанавливается имя файла по умолчанию.

При открытии файла используется метод LoadFile() компонента RichTextBox.

При сохранении используется метод SaveFile().

Если файл не был сохранен ранее, команда Save вызывает Save As для выбора имени файла.

## Форма About

Создана отдельная форма AboutBox1 с информацией о программе:
- Название программы
- Версия
- Копирайт
- Описание
- Кнопка OK для закрытия

Форма использует TableLayoutPanel для размещения элементов.
