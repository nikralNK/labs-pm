# Лабораторная работа 20 - Вычисление функции с поиском экстремумов

## Описание задачи

Создано приложение для вычисления значения функции на отрезке [xn; xk] с шагом xh и нахождения минимального и максимального значений функции.

## Формулы функций

### Функция 1 (при x <= 0)
```
f = |x|⁵ * ctg(x + 2)
```

### Функция 2 (при 0 < x <= a)
```
f = (5*x + x²) / (x² + 3)³
```

### Функция 3 (при x > a)
```
f = sin²(x + 3) / (x⁵ - ctg(π * x³))
```

## Параметры по умолчанию

- Начальное значение: xn = 3.35
- Конечное значение: xk = 36.26
- Шаг: xh = 0.2

## Элементы интерфейса

**Поля ввода:**
- `txtA` - параметр a
- `txtXn` - начальное значение x (по умолчанию 3.35)
- `txtXk` - конечное значение x (по умолчанию 36.26)
- `txtXh` - шаг изменения x (по умолчанию 0.2)

**Кнопки:**
- `btnCalculate` - вычисление функции на всем отрезке
- `btnClear` - очистка результатов
- `btnExit` - выход из приложения

**Вывод результатов:**
- `lstResults` - ListBox для отображения таблицы значений x и f(x)
- `txtMin` - поле для отображения минимального значения функции
- `txtMax` - поле для отображения максимального значения функции

## Логика работы

### Метод CalculateFunction

Принимает значения x и a, возвращает значение функции f(x).

Логика выбора функции:
1. Если x <= 0, вычисляется функция 1
2. Если 0 < x <= a, вычисляется функция 2
3. Если x > a, вычисляется функция 3

Используемые математические функции:
- `Math.Pow(x, n)` - возведение в степень
- `Math.Abs(x)` - модуль числа
- `Math.Tan(x)` - тангенс (ctg = 1/tan)
- `Math.Sin(x)` - синус
- `Math.PI` - константа π

### Метод btnCalculate_Click

Основной обработчик вычисления с использованием цикла While:

1. Считывает параметры a, xn, xk, xh
2. Очищает ListBox и поля результатов
3. Инициализирует переменные:
   - x = xn (начальное значение)
   - fMin = максимальное значение double (для поиска минимума)
   - fMax = минимальное значение double (для поиска максимума)
   - xMin, xMax - для запоминания точек экстремумов

4. Добавляет заголовок таблицы в ListBox

5. **Цикл While** (x <= xk):
   - Вычисляет f = CalculateFunction(x, a)
   - Добавляет строку с x и f(x) в ListBox
   - Проверяет, является ли f минимальным значением
   - Проверяет, является ли f максимальным значением
   - Увеличивает x на шаг xh

6. После завершения цикла выводит найденные экстремумы:
   - Минимум: f(xMin) = fMin
   - Максимум: f(xMax) = fMax

### Метод btnClear_Click

Очищает ListBox и поля с экстремумами.

### Метод btnExit_Click

Закрывает приложение.

## Поиск экстремумов

Алгоритм поиска минимума и максимума:

**Инициализация:**
- fMin = double.MaxValue (самое большое возможное значение)
- fMax = double.MinValue (самое маленькое возможное значение)

**В цикле:**
- Если текущее f < fMin, то обновляем fMin и запоминаем xMin
- Если текущее f > fMax, то обновляем fMax и запоминаем xMax

Таким образом, после прохода всех точек мы получаем минимальное и максимальное значения функции на заданном отрезке.

## Формат вывода

Результаты выводятся в виде таблицы:
```
     x          |        f(x)
----------------|----------------
           3.35 |      0.123456
           3.55 |      0.234567
           ...
```

Экстремумы выводятся в формате:
```
f(3.75) = 0.123456
```

## Обработка ошибок

При возникновении ошибок вычисления (деление на ноль, некорректный ввод) выводится сообщение "Ошибка вычисления".
