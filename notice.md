# Лабораторная работа 17

## Задание 1.1: Кнопки для изменения цвета формы

### Как работает код

Создано четыре кнопки с текстом "красный", "зеленый", "синий" и "желтый".

Каждая кнопка имеет обработчик события Click:
- `btnRed_Click` - меняет цвет фона формы на красный
- `btnGreen_Click` - меняет цвет фона формы на зеленый
- `btnBlue_Click` - меняет цвет фона формы на синий
- `btnYellow_Click` - меняет цвет фона формы на желтый

Изменение цвета происходит через свойство `BackColor` формы. Используются стандартные цвета из класса `Color`.

## Задание 1.2: Кнопки для изменения текста метки

### Как работает код

Создано две кнопки с текстом "привет" и "до свидания", а также метка для отображения текста.

Обработчики событий Click:
- `btnHello_Click` - меняет текст метки на "привет"
- `btnGoodbye_Click` - меняет текст метки на "до свидания"

Также создан обработчик события `Form1_Load`, который выполняется при загрузке формы:
- Устанавливает цвет фона формы в светло-серый
- Устанавливает текст метки в "Начало работы"

Изменение текста происходит через свойство `Text` метки.

## Задание 2: Вычисление математических функций

### Как работает код

Создана группа элементов для вычисления двух математических функций.

Элементы интерфейса:
- Три текстовых поля для ввода значений X, Y, Z
- Кнопка "Вычислить"
- Две метки для отображения результатов

### Формулы

Функция 1:
```
t = (2 * cos(x - π/6) / (0.5 + sin²(y))) * (1 + z² / (3 - z²/5))
```

Функция 2:
```
u = ∛(8 + |x-y|² + 1) / (x² + y² + 2) - e^|x-y| * (tan²(z) + 1)^x
```

### Логика работы

При нажатии кнопки "Вычислить":

1. Считываются значения из текстовых полей и преобразуются в числа типа `double`
2. Вычисляется первая функция `t`:
   - `Math.Cos` - вычисление косинуса
   - `Math.PI` - константа π
   - `Math.Sin` и `Math.Pow` - вычисление синуса в квадрате
3. Вычисляется вторая функция `u`:
   - `Math.Pow(..., 1.0/3.0)` - кубический корень
   - `Math.Abs` - модуль числа
   - `Math.Exp` - экспонента
   - `Math.Tan` - тангенс
4. Результаты отображаются в метках с точностью 6 знаков после запятой

Если при вводе данных возникает ошибка (некорректный формат числа), выводится сообщение "Ошибка ввода данных".
